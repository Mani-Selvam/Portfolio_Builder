Portfolio Website Generator

A comprehensive system for collecting and managing client portfolio submissions with file uploads and admin dashboard.

Client Submission Form Admin Dashboard

Features

Client Submission Form

Comprehensive form for portfolio submissions with personal details, education, work experience, and project information

File Upload System

Upload resumes (PDF), profile photos, and project screenshots with secure storage

Admin Dashboard

Manage submissions, update status, view detailed information, and export data

Database Integration

PostgreSQL database with Drizzle ORM for reliable data storage and management

Deployment Note: This is a static preview. For full functionality including form submissions, file uploads, and admin features, the backend server and database need to be deployed separately (Railway, Render, or Vercel recommended).
